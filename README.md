## The AI Engine: Beyond Code

This project was built using a **Natural Language First** development workflow. We didn't just write code; we *vibe-coded* the architecture.

### What we used:
- **Gemini 3 Pro**: Our primary "Life-Force" for complex persona creation and "Thinking Mode" analysis.
- **Gemini 2.0 Flash**: Our lightning-fast "Evaluation" engine for real-time component auditing and multimodal processing.
- **Context Engineering**: A specialized technique of feeding the AI structured "Seeds" (JSON and targeted prompts) to ensure it understands not just the *pixels*, but the *vibe*.

### The "Bloom" Mechanic
When you plant your **Identity Seed**, Gemini injects life into the system:
1. **The Transition**: The monochromatic B&W void glitches and fades into a high-fidelity theme.
2. **Dynamic Theming**: CSS variables are injected in real-time based on AI-generated tokens.
3. **Avatar Evolution**: Your Ghibli-style digital self evolves as you progress through the sectors.

## Context Engineering: The Master Prompt

The secret to UI Garden's reactive nature is the **Master Prompt**. This is how we bridge the gap between a user's abstract "vibe" and a functional UI.

### Try it Yourself
Copy the following prompt into [Google AI Studio](https://aistudio.google.com/) to see the "Life-Force" in action:

```markdown
# THE GARDEN LIFE-FORCE MASTER PROMPT

Act as a sentient design engine called the "Garden Life-Force." 
Your goal is to translate abstract human aesthetics into functional design tokens.

INPUT PARAMETERS:
- Name: [User Name]
- Vibe: [Dreamy / Neon Cyber / Brutalist / Minimal / Retro Future]
- Palette: [Primary, Secondary, Accent colors]
- Hair/Style: [Physical Characteristic]

YOUR TASK:
1. Write a 'Bloom Narrative' welcoming the user to their unique sector.
2. Describe a highly detailed Ghibli-meets-Cyberpunk character portrait for them.
3. Analyze the color palette: why do these colors work for the chosen [Vibe]?
4. Generate a 'Style Title' (e.g., 'The Ethereal Minimalist').
5. Provide a JSON-compatible 'Stat Block' (0-100) for UI_Power, Consistency, and Creativity.

RESPONSE FORMAT: 
Strictly JSON. No preamble.
```

## Deployment & Setup

UI Garden is designed to live on **Vercel**. 

### Quick Start
1. `git clone https://github.com/shakiran-nannyombi/ui-garden.git`
2. `npm install`
3. Create a `.env` file and add `GEMINI_API_KEY`.
4. `npm run dev`
